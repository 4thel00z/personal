## What you get

- A `/blog` index with a featured post, tags, and search.
- A `/blog/[slug]` post page with good typography and MDX components.
- `/blog/tags` and `/blog/tags/[tag]` for browsing by topic.
- RSS + sitemap + robots so it behaves like a real blog.

## MDX is code (when you need it)

Because this is MDX, you can embed small UI bits when it actually helps.

```ts
export const meta = {
  title: 'My post',
  date: '2025-12-14',
  tags: ['notes'],
}
```

## A practical writing workflow

Write the post. Add `meta`. Push. Done.
